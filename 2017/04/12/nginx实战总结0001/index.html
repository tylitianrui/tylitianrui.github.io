nginx主要分为四部分：

  • nginx 二进制文件
  • nginx配置文件 nginx.conf等
  • access.log 接入日志
  • error.log 错误日志

配置文件控住nginx的行为，一般情况下，linux默认的配置文件的路径为 /etc/nginx/nginx.conf 注：在不同的系统中，配置文件的路径可能会不同

1.2结构总结#

由上可以看出，nginx的配置文件，是依照 块进行划分的，例如:server、 http、event等。但他们之间有什么区别与联系呢？
nginx主要分为6块：

  • 全局模块，例如 user nobody; worker_processes 1;
  • events 块
  • http 块
    • server 块
      • location 块
    • upstream 块

一个http块里面可以有多个server，一个server里面可以有多个location
